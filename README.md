# GVSRobotics website starter

This folder contains a simple GitHub Pages landing page for the GVSRobotics organization.

## What it includes
- An overview section
- Members and owners section
- A public repositories section that pulls data from the GitHub API
- A work-in-progress section
- A contact section

## How to publish it
1. Create or use a public repository for the site, such as `GVSRobotics.github.io`.
2. Copy the contents of this folder into the repository root.
3. Enable GitHub Pages in the repository settings.
4. Choose the `GitHub Actions` deployment source.

## Customization tips
- Replace the placeholder contact email with a real address.
- Update the member cards in `script.js` with real names and GitHub links.
- Add or remove sections based on the organization’s needs.
